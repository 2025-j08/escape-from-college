 # js
 
 役割: ゲームのJavaScriptソースを格納します。
 
 説明:
 - ゲームのエントリポイント（例: `main.js`）やシーン管理、入力処理、ゲームループ、アセット読み込みなどの実装を配置します。
 - GitHub Pagesへそのまま配信できるよう、可能であればビルド不要（生のES5/ES6）で動作するコード構成を推奨します。ESモジュールを使う場合はHTML側で`<script type="module">`を指定してください。
 - ファイル例: `main.js`, `scene/`（画面ごとのロジック）, `lib/`（共通ユーティリティ）
 
 備考: 大きな外部依存やビルド出力を追加する場合は、READMEにビルド手順を追記してください。
