# escape-from-college
The 密室からの脱出 ～まやかしの専門学校～

このリポジトリは、HTML/CSS/JavaScriptで作るシンプルなブラウザゲームプロジェクトです。
静的サイトとしてGitHub Pagesで配信できる構成を想定しています。

ディレクトリの役割概要:

- `index.html`, `css/`, `js/`: ゲームの公開用ページとソース（HTML/CSS/JS）。ビルド不要でそのまま配信できる構成を推奨します。
- `asset/`: 画像・音声・フォント等のアセットを格納します（`asset/images`、`asset/audio`、`asset/fonts`）。
- `data/`: レベル定義や設定など、ゲームで読み込むJSON等の静的データを格納します。
- `docs/`: 仕様書、デザインノート、アセットの帰属情報などの補足ドキュメントを置きます。

不要なものの補足:

現時点でリポジトリ内にビルド出力や依存管理の大量ファイルは含まれていないため、特に削除が必要な不要物は見当たりません。
将来的にビルドツールやパッケージ管理を追加する場合は、`dist/`や`node_modules/`等を`.gitignore`へ追加してください。

今後の推奨:

- 小さなプロジェクトのまま配信する場合は、できるだけ外部ビルドを使わずに動作させるとGitHub Pagesでの配信が容易になります。
- ドキュメントは`docs/README.md`に設計意図や画面遷移図をまとめておくと保守が楽になります。
